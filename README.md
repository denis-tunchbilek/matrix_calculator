# Matrix Calculator (C++23)

Team project for a console matrix calculator in modern C++.

## Planned structure
- src/core — matrix model, linear algebra, constants, errors
- src/io — matrix input and output
- src/util — helper value types and printers
- src/report — report steps and runner
- src/app — application entry point
- tests — unit tests

## Planned goals
- C++23
- OOP architecture
- abstract base class + polymorphism
- CMake build
- unit tests
- modern C++ features:
- std::unique_ptr
- std::shared_ptr
- std::optional
- std::variant / std::visit
- RAII
- constexpr
- exceptions
